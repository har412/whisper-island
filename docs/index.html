Build & Hack the Code

WhisperIsland is 100% open source. Clone, modify, and build it natively on your machine:

# 1. Clone the repository
git clone https://github.com/har412/whisper-island.git

# 2. Go to project and install dependencies
cd whisper-island
npm install

# 3. Launch application
npm start

# 4. Compile standalone Windows exe
npm run build:exe
